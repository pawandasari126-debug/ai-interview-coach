# 🤖 AI Interview Coach

An interactive AI-powered interview practice application with real-time feedback and performance tracking.

## 🚀 Features
- Chat-based interview experience (like real interviewer)
- Dynamic follow-up questions
- Real-time scoring and feedback
- Progress tracking during interview
- Final performance analysis

## 🛠️ Tech Stack
- Python
- Streamlit

## ▶️ Run Locally
```bash
pip install streamlit
python -m streamlit run ai_interviewer.py
