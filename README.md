# 🎤 AI Interview Coach PRO

An AI-powered interview preparation platform built with **Python**, **Streamlit**, and **Groq AI** that helps users practice technical and HR interviews with real-time AI feedback, resume analysis, coding challenges, voice interviews, emotion detection, and detailed performance analytics.

---

## ✨ Features

### 📄 AI Resume Analyzer
- ATS-style resume analysis
- Resume score
- Strengths & weaknesses
- Missing skills detection
- Suitable job role recommendations

### 🎤 AI Interview Simulator
- HR Interview
- Data Analyst Interview
- Python Developer Interview
- AI/ML Engineer Interview
- Dynamic AI-generated questions
- Instant AI evaluation
- Question-wise scoring
- Detailed feedback

### 🎙️ Voice Interview
- Speech-to-text recognition
- AI evaluation of spoken answers
- Communication analysis
- Voice interview practice

### 💻 Coding Interview
- Python coding questions
- SQL coding questions
- Multiple difficulty levels
- AI code evaluation
- Coding performance tracking

### 📹 Webcam Emotion Analysis
- Real-time emotion detection
- Confidence estimation
- Interview behavior analysis
- Live webcam feedback

### 📈 Reports Dashboard
- Overall interview score
- Interactive charts
- Performance analytics
- Candidate readiness gauge
- AI recommendations
- PDF interview report generation

# 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/pawandasari126-debug/ai-interview-coach.git
```

### Go to the project folder

```bash
cd ai-interview-coach
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Create a secrets file

Create the following file:

```
.streamlit/secrets.toml
```

Add your Groq API Key:

```toml
GROQ_API_KEY="YOUR_GROQ_API_KEY"
```

### Run the application

```bash
streamlit run app.py
python -m streamlit run app.py
```
